LA DESCRIPTION:

HOLA est un modèle de site Web vCard moderne et élégant. Conçu pour être un CV, vCard, portefeuille
modèle, c'est le modèle parfait pour les concepteurs créatifs, les développeurs, les pigistes, les photographes
ou toute profession créative. Il est entièrement réactif et prêt pour la rétine / hi-dpi. Il aura fière allure
sur tous les appareils, du bureau aux téléphones portables. Il a des modèles de page de blog, formulaire de contact de travail,
section de portefeuille élégant et d'autres fonctionnalités que vous trouverez uniquement sur les modèles premium. Construit avec
code propre et organisé, ce modèle est également très facile à personnaliser.
